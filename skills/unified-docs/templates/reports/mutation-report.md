# Mutation Report

Use for Maintain mode after changing files, or for Create mode only when the user explicitly asks for a report or summary in addition to the authored document.

```markdown
Mode: [Create | Maintain]
Status: [completed | partial | blocked]

Changed files:
- `[path]` — [what changed]

Why / key decisions:
- [Reason for the change, source rationale, classification, lifecycle, or cascade decision]

Validation:
- [script/check/manual validation]

Followups:
- [Remaining review or none]
```
